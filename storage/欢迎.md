# 欢迎使用 nas-md

nas-md 是一个轻量级的 Markdown 笔记服务，专为 NAS 和自托管场景设计。

## 快速上手

### 打开文件

在左侧边栏选择一个挂载目录，点击 `.md` 文件即可打开编辑。

### 编辑模式

顶部工具栏可切换三种模式：

- **即时渲染** —— 左侧写 Markdown，右侧实时预览
- **分屏预览** —— 同屏显示源码和渲染结果
- **所见即所得** —— 像 Word 一样直接编辑排版

### 保存

- **自动保存**默认开启，编辑后自动保存到服务器
- 也可手动点击「保存」按钮或按 `Ctrl+S`

## 挂载目录

### 宿主机目录（Admin）

如果你通过 Docker 部署并在 `volumes` 中挂载了宿主机目录到 `/mnt/` 下，访问 `/admin` 即可看到并编辑这些目录中的文件。

### 浏览器挂载

在欢迎页点击「浏览…」按钮，选择本地目录即可挂载：

- HTTPS 环境下支持读写
- HTTP 环境下仅只读

### 手动输入路径

在欢迎页输入框中输入服务器上的目录路径，点击「挂载」。

## 图片引用

Markdown 中使用相对路径引用的图片会自动解析显示。例如：

```markdown
![示例图片](photo.png)
```

只要 `photo.png` 和当前 `.md` 文件在同一目录下，就能正确显示。

## 快捷键

| 快捷键 | 操作 |
|--------|------|
| `Ctrl+K` | 搜索文件 |
| `Ctrl+S` | 保存 |
| `Ctrl+N` | 新建笔记 |
| `Ctrl+B` | **粗体** |
| `Ctrl+I` | *斜体* |
| `Ctrl+~` | 切换侧边栏 |

## 更多功能

- **全文搜索** —— 侧边栏搜索框，毫秒级检索
- **知识图谱** —— 可视化笔记之间的链接关系
- **数据看板** —— 笔记数、任务完成率、标签统计
- **暗色模式** —— 顶部月亮图标一键切换
