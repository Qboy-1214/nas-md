# 欢迎使用 nas-md

> **点击左下角「挂载文件夹」按钮，选择本机目录即可开始使用。**
>
> Docker 部署时，宿主机目录会自动出现在侧边栏。

---

## 快速上手

### 打开文件

在左侧边栏选择一个挂载目录，点击 `.md` 文件即可打开编辑。

### 创建文件

在可读写的目录下，点击目录名右侧的 **+** 按钮即可新建文件夹或 Markdown 文件。

### 编辑模式

顶部工具栏可切换三种模式：

- **即时渲染（IR）** —— 输入 Markdown 标记后立即渲染，适合习惯 Markdown 语法的用户
- **分屏预览（SV）** —— 左侧写 Markdown 源码，右侧实时预览渲染结果
- **所见即所得（WYSIWYG）** —— 像 Word 一样直接编辑排版，适合不熟悉Markdown的用户

### 保存

- **自动保存**默认开启，编辑后自动保存到当前目录
- 也可手动点击「保存」按钮或按 `Ctrl+S`

### 重命名

点击顶部工具栏的 **重命名** 按钮，输入新名称即可。重命名后侧边栏和路径会即时更新。

### 删除

点击顶部工具栏的 **删除** 按钮，确认后即可删除当前文件。此操作不可撤销。

### 移动与复制

在侧边栏中**拖拽**文件或文件夹到目标目录：

- 拖到**同一挂载点**内 → 移动
- 拖到**不同挂载点** → 弹框选择「移动」或「复制」
- 目标位置有同名文件时，自动加时间戳后缀重命名，不会覆盖

### 从浏览器外导入

直接从系统文件管理器**拖拽 `.md` 文件**到侧边栏的任意目录，即可导入。非 `.md` 文件会被拦截并提示。

## 挂载目录

### 宿主机目录

- **公开目录** —— 所有用户可见可读写（通过 `PUBLIC_MOUNTS` 环境变量配置）

### 浏览器挂载

点击左下角「挂载文件夹」按钮，选择本地目录即可挂载：

- HTTPS 环境下支持读写
- HTTP 环境下仅只读

## 图片引用

Markdown 中使用相对路径引用的图片会自动解析显示。例如：

```markdown
![示例图片](photo.png)
```

只要 `photo.png` 和当前 `.md` 文件在同一目录下，就能正确显示。

## 快捷键

| 快捷键 | 操作 |
|--------|------|
| `Ctrl+K` | 搜索文件 |
| `Ctrl+S` | 保存 |
| `Ctrl+N` | 新建笔记 |
| `Ctrl+B` | **粗体** |
| `Ctrl+I` | *斜体* |
| `Ctrl+~` | 切换侧边栏 |

## 更多功能

- **全文搜索** —— 侧边栏搜索框，毫秒级检索
- **暗色模式** —— 顶部月亮图标一键切换
- **双向链接** —— 使用 `[[文件名]]` 语法链接笔记，自动追踪反向链接
